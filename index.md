## Welcome to Field Notes

Welcome to **Field Notes**, a collection of observations drawn from my own long-term records of the world around me.

The site brings together several small observation projects that document different aspects of the local environment. These records are collected over time using tools I’ve developed for my own use, and are published here as structured reports, charts, and datasets.

You’ll find reports and summaries covering:

- **Wildlife sightings** and natural history records
- **Aircraft observations**
- **Microscopy investigations** and photomicrographic plates
- **Weather station data** recorded by a local environmental monitoring system

Together, these reports form a set of **personal field notes**, turning day-to-day observations into organised datasets and visual summaries.

---

## Observation tools

The data behind these reports is collected using a small set of applications developed for my own use:

- [**Nature Recorder**](https://github.com/davewalker5/NatureRecorderPy)  
  Used to log wildlife sightings and build long-term natural history records.

- [**Flight Recorder**](https://github.com/davewalker5/FlightRecorder)  
  Used to capture and analyse aircraft observations.

- [**Raspberry Pi Weather Station**](https://github.com/davewalker5/RPi-WeatherStation)  
  A small environmental monitoring system that records local weather conditions.

In addition, the ADS-B tracking tools can also be used for live aircraft monitoring:

- [**ADS-B BaseStation Reader**](https://github.com/davewalker5/ADS-B-BaseStationReader)

## Explore the reports

Across the site you’ll find automatically generated summaries and visualisations derived from these datasets, including:

- trend reports  
- annual heatmaps  
- species richness summaries  
- aircraft traffic analysis  
- environmental monitoring reports

Each report provides both **visual charts** and **downloadable data**, allowing the underlying observations to be explored in more detail.

---

## Use the tools yourself

All of the applications used to generate these records are **open source**. If you’re interested in keeping track of your own observations, you’re very welcome to explore the code and adapt the tools for your own projects.
