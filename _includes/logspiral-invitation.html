LogSpiral Shell Modeller

Exploring shell growth, coiling geometry, and biological morphospace

The shell models shown throughout these investigations were generated using a small procedural modelling system based on logarithmic spirals and swept apertures.

By varying only a handful of interpretable growth parameters — including expansion rate, aperture geometry, ornamentation, and coiling behaviour — the system can generate shell forms reminiscent of nautiluses, ammonites, ramshorn snails, and more speculative morphologies.

The project is intended not simply as a graphics exercise, but as an exploration of computational natural history: investigating how complex biological form can emerge from relatively simple developmental rules.

The application, presets, example notebooks, and instructions for running the modeller are provided in the GitHub repository.
