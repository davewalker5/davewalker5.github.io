# The Example Shell Presets

The shell presets presented on subsequent pages explore how relatively small changes in growth and aperture geometry can generate dramatically different shell morphologies using variations in:

- Logarithmic growth rate
- Aperture scaling
- Coiling geometry
- Shell inflation
- Ornamentation (ribbing)
- Aperture flare
- Shell wall thickness

Although highly simplified, these presets demonstrate how a relatively small set of growth parameters can produce shell forms reminiscent of real molluscan and cephalopod shells.
