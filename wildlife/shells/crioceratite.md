# Crioceratite-Like Shell

{% assign assets_path = site.assets_url | append: "/images/modelling/shells/" %}
{% include image-block.html images=page.images assets=assets_path %}

The crioceratite-like preset explores an open-coiled ammonoid morphology derived from the same underlying logarithmic spiral growth framework as the ammonite preset.

Unlike the tightly coiled ammonite form, successive whorls remain separated, producing a shell with a more open and loosely wound appearance. Strong transverse ribbing and accelerated shell expansion further emphasise the distinctive character of these heteromorph ammonoids.

Characteristics:

- Open coiling with separated whorls
- Strong transverse growth ribs
- Moderate to rapid shell expansion
- Chambered internal structure
- Visible siphuncle following the spiral path

The resulting morphology resembles fossil crioceratites and related open-coiled heteromorph ammonites, which occupy an intermediate position between conventional planispiral ammonites and the more extreme uncoiled ammonoid forms.

From a modelling perspective, the crioceratite provides an interesting demonstration that substantial morphological change can emerge without introducing a new growth law. The shell remains a logarithmic spiral generated using the same underlying framework as the ammonite model, with the overall form arising primarily from changes in whorl interaction, shell expansion and ornamentation.

The preset illustrates how relatively modest adjustments to growth parameters can transform a tightly coiled ammonite into a recognisably distinct open-coiled morphology while retaining the characteristic internal architecture of a chambered cephalopod shell.

Transparent rendering additionally reveals:

- Internal chamber spacing
- Septal placement
- Siphuncle trajectory
- The persistence of ammonoid internal structure despite the more open shell form

providing a simplified visualisation of the internal anatomy shared by both coiled and open-coiled ammonoids.
